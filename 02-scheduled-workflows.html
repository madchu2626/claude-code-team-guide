Setting Up Scheduled Workflows — What to Say to Claude Code

Jump to: Key Terms Before You Start Prompt 1: Setup Prompt 2: Customize Prompt 3: Schedule Prompt 4: Cron Job Prompt 5: Show Setup Prompt 6: Fix Issues Decision Guide Common Schedules
How this works: Once Claude Code is installed (see Setup Guide), you just tell it what you want. Below are the exact prompts to copy-paste into Claude Code. Claude Code will do the technical work for you.

You can use the Claude desktop app or Terminal — both work the same way. The desktop app's code window is usually easier.

Key Terms on This Page

TermWhat it means
GitHub Actions Runs tasks on a schedule on GitHub's servers — works even when your computer is off. Best for most people.
Cron Job Runs tasks on a schedule on your Mac — but only when your computer is on and awake. If your Mac is asleep or off, it skips that run. Use this if you don't want to deal with GitHub.
Workflow A set of instructions that runs automatically on a schedule. Could be a GitHub Action or a cron job — both are types of workflows.

Before You Start

Option A: Using the Claude Desktop App (Recommended)

  1. Open the Claude desktop app
  2. Open the code window (look for the terminal/code icon)
  3. Navigate to your project folder by typing: cd ~/Desktop/my-project
  4. Copy-paste the prompts below

Option B: Using Terminal

  1. Open Terminal (Command+Space → type Terminal → Enter)
  2. Navigate to your project folder:
cd ~/Desktop/my-project
  1. Start Claude Code:
claude
  1. Copy-paste the prompts below
Tip: If Claude keeps asking "Allow this action?" before every step, type /permissions and select "Allow All". This lets Claude work without pausing for approval each time.

PROMPT 1: "Set up everything I need for scheduled workflows"

Copy-paste this into Claude Code:

I want to set up automated workflows that run on a schedule (like daily, or every few hours). This project is a git repo connected to GitHub.

Please help me:

1. Check if this repo has GitHub Actions set up already. If not, set it up.
2. Check if I have the Claude GitHub App installed. If not, walk me through installing it step by step.
3. Make sure I have an ANTHROPIC_API_KEY stored as a GitHub Secret. If not, walk me through how to add one step by step.
4. Create a GitHub Actions workflow file that runs on a schedule.

For now, make it a simple daily workflow that runs at 9am UTC. We'll customize what it does in the next step.

Go step by step. Wait for my confirmation before making any changes. Explain everything in plain English — I'm not a developer.

PROMPT 2: "Make it do [your actual task]"

After the setup is done, tell Claude what you actually want the scheduled workflow to do. Here are examples:

Example — Daily report:

Now update the scheduled workflow so it does this every day at 9am UTC:
- Look at all the files in this project
- Generate a summary of what changed in the last 24 hours
- Post the summary as a GitHub Issue

Keep the workflow simple and add a budget limit of $2 per run.

Example — Run a script every 6 hours:

Now update the scheduled workflow so it runs every 6 hours and does this:
- Run the script at scripts/check-data.py
- If the script fails or finds errors, create a GitHub Issue with the error details

Keep it simple and add a budget limit of $1 per run.

Example — Weekly cleanup:

Now update the scheduled workflow so it runs every Monday at 8am UTC:
- Check for any open GitHub Issues older than 30 days
- Add a comment on each one asking if it's still relevant
- Create a summary Issue listing all stale issues

Keep it simple and add a budget limit of $2 per run.

PROMPT 3: "Change the schedule"

If you want to change when it runs, just say so:

Change the scheduled workflow to run every 4 hours instead of daily.
Change the scheduled workflow to run Monday through Friday at 9am UTC only (not weekends).
Change the scheduled workflow to run twice a day — at 9am and 5pm UTC.

PROMPT 4: "Set up a local cron job instead"

If you want something to run on YOUR computer (not on GitHub), use this:

I want to set up a cron job on my Mac that runs automatically on a schedule.

Here's what I want:
- Every day at 9am, run Claude Code with this task: "[DESCRIBE YOUR TASK HERE]"
- Save the output to a log file so I can check it later
- Make sure my API key is loaded properly
- Use the --dangerously-skip-permissions flag so it doesn't get stuck waiting for approval

Set this up for me step by step. Explain what a cron job is in simple terms. Show me how to check if it's running and how to stop it if I need to.

Important: I'm not a developer, so please explain everything in plain English.
Note about cron jobs:
- Your computer needs to be ON and AWAKE for cron jobs to run
- If your Mac is asleep or off at the scheduled time, it will skip that run
- GitHub Actions (Prompts 1-3 above) run on GitHub's servers, so they work even when your computer is off
- For most people, GitHub Actions is the better choice

PROMPT 5: "Show me what's already set up"

If you forget what workflows you have:

Show me all the scheduled workflows set up in this project. For each one, tell me:
- What it does (in plain English)
- When it runs
- Where I can see the results

Explain it simply.

PROMPT 6: "Something is broken"

If a workflow stops working:

The scheduled workflow in this project seems broken. Can you:
1. Check the GitHub Actions tab for any failed runs
2. Tell me what went wrong in plain English
3. Fix it if you can, or tell me what I need to do

Explain everything simply — I'm not a developer.

Common Schedules (for reference)

When Claude asks you what schedule you want, here are common options:

ScheduleHow to say it
Every day at 9am"Run daily at 9am UTC"
Every weekday at 9am"Run Monday to Friday at 9am UTC"
Every 6 hours"Run every 6 hours"
Every hour"Run every hour"
Every Monday at 8am"Run weekly on Monday at 8am UTC"
Twice a day"Run at 9am and 5pm UTC"
What's UTC?
UTC is a universal time zone. If you're in Pacific Time, UTC is 8 hours ahead. So "9am UTC" = 1am Pacific = 4am Eastern. Just tell Claude what time you want in YOUR timezone and it will convert it.
