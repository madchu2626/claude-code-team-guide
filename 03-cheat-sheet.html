Claude Code — Cheat Sheet

Print this page. Keep it at your desk.

Open Claude Code

  1. Open Terminal (Command+Space → type Terminal → Enter)
  2. Go to your project folder:
cd ~/Desktop/your-project-name
  1. Type claude and press Enter

Everyday Commands

What you wantWhat to type in Terminal

Exit / stopPress Escape, type /exit, press Enter
Force stopCtrl + C (press twice if needed)

Check healthclaude doctor
Check loginclaude auth status
Re-loginclaude auth login

What you wantWhat to type
Ask Claude somethingJust type your question and press Enter
Cancel what Claude is doingPress Escape
Clear the screenType /clear and press Enter

Something Not Working?

  1. Run claude doctor — it will tell you what's wrong
  2. Try claude update — you might need a newer version
  3. Try claude auth login — your login might have expired
  4. Close Terminal and reopen it — sometimes that fixes things
  5. Still stuck? Screenshot the error and send it to your team lead
